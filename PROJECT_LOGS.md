## v0.3 (2025-01-11) — Markdown рендеринг

### Проблема
Markdown файлы материалов курса отображались как неотформатированный текст:
```
# Модуль 1: Подготовка рабочей среды
## Цели модуля
- Освоить...
```
Без заголовков, отступов, таблиц и блоков кода.

### Решение
1. Установлен npm пакет `marked` для конвертации markdown в HTML
2. Обновлён `server.js` для обработки `.md` файлов:
   - Чтение markdown файла
   - Конвертация через `marked.parse()`
   - Обёртка в HTML шаблон со стилями
3. Создан `css/markdown.css` с полным оформлением:
   - Красивые заголовки с подчёркиваниями
   - Отступы между элементами
   - Стили для таблиц, кодблоков, списков
   - Адаптивность для мобильных

## v0.2 (2025-01-11) — Исправление ссылок

### Проблема 1: Симлинки
Файлы упражнений и решений не были доступны по URL:
```
 course-materials → ../course/materials  (существовал)
 course-exercises → не существовал
 course-solutions → не существовал
```

### Решение
Созданы симлинки для всех папок курса:
```bash
ln -sf ../course/materials course-materials
ln -sf ../course/exercises course-exercises  
ln -sf ../course/solutions course-solutions
```

---

### Проблема 2: Query String в URL
При переходе `exercises.html?module=01` сервер искал файл `exercises.html?module=01` вместо `exercises.html`.

### Решение
В `server.js` добавлено разделение URL:
```javascript
const [pathOnly] = req.url.split('?');
let filePath = path.join(STATIC_DIR, pathOnly === '/' ? 'index.html' : pathOnly);
```

---

### Проблема 3: Неправильные пути к файлам упражнений
В `exercises.html` ссылки генерировались с `*` вместо реальных имён файлов:
```javascript
// Было (не работало):
href="course/exercises/module-${moduleId}/exercise-${ex.id}-*.md"

// Стало (работает):
href="course-exercises/module-${moduleId}/${ex.file}"
```

### Решение
Создана структурированная конфигурация упражнений в `exercises.html`:
```javascript
const exercises = {
    '01': [
        { id: '01-01', file: 'exercise-01-01-terminal-basics.md', solution: 'solution-01-01-terminal-basics.md' },
        { id: '01-02', file: 'exercise-01-02-vscode-setup.md', solution: 'solution-01-02-vscode-setup.md' }
    ],
    // ...
};
```
